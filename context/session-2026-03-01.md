## What This Is
Execution infrastructure for AI agents to trade crypto autonomously on Base.

- Agent holds a wallet
- Agent calls the cluster JSON-RPC endpoint to execute swaps
- Access gated by OperatorNFT (tiered: 1=basic, 2=pro, 3=operator)
- Agent pays subscription via SubscriptionKeeper (autonomous, no human)
- Swap fees: 0.1% take-rate to fee_recipient before routing

**The primitive that doesn't exist:** agent-pays-agent. Agent holds keys, pays fees, gets execution — no human touchpoint.
